## 🚀 Возможности

### **Игрок & Статистика**
- **Деньги / XP / Уровень / Тикеты / Фишки**: Мгновенное изменение ключевых значений прогресса.
- **Бесконечные деньги (Переключатель)**: Заморозка денег на высоком значении.
- **Достижения**: Выдать или сбросить достижения.
- **Редактор статистики**: Клики, открытые кейсы, время игры и т.д.
- **Косметика профиля**: Локальная смена аватарки + цветной ник (клиентская часть).

### **Инвентарь**
- **Выдать все предметы (Оптимизировано)**: Добавляет по 1 экземпляру каждого предмета.
- **Дублировать инвентарь**: Клонирует все предметы.
- **Дюп конкретного предмета**: Выбор предмета для дублирования.
- **Удалить фейк-предметы**: Убирает сломанные/неизвестные предметы.
- **Очистить инвентарь**: Удаляет все предметы.
- **Выдать всё**: Кейсы / Ключи / Ножи / Пушки / Перчатки.
- **Выдать конкретный предмет**: Добавление по точному названию.
- **Купить много**: Массовая покупка предметов.

### **Функции игры**
- **Миссии**: Завершить / получить новые / удалить / сбросить таймер миссий.
- **Бесплатный магазин (Переключатель)**: Все предметы стоят $0.
- **Автокликер (Переключатель)**: Очень высокая скорость кликов.
- **Убрать лимит кликов (Установка)**: Отключает ограничение анти-автокликера.
- **Фикс мульти-вкладки**: Убирает предупреждение “Multiple tabs detected”.

### **Сервер / Онлайн**
- **Очистить Online Coinflip**: Очищает локальный список онлайн-коинфлипов.
- **Анонимный ID (Переключатель)**: Временный случайный ID.
- **Победа в Coinflip (Визуально/Локально)**: Визуальная победа (на сервер не влияет).
- **Редактор джекпота (Заглушка)**: Открывает отладочную информацию.

### **Мини-игры & RNG**
- **Симулятор побед**: Начисляет фишки для Crash, Roulette, Mines, Slots.
- **Подкрутка RNG**: Всегда высокий шанс (0.99) или низкий (0.001), можно сбросить.

### **Автоматизация 🤖**
- **Бот кейсов**: Авто-открытие кейсов до нужного результата.

### **Утилиты**
- **Получить сохранение / Загрузить сохранение**: Экспорт/импорт строки сохранения.
- **Полный сброс**: Полная очистка локальных данных.
- **Изменить заголовок вкладки**: Смена title браузера.
- **Скрыть Predator**: Прячет меню Predator (если есть).
- **Скрипт-экзекутор**: Выполнение собственного JS-кода.
- **Проверка версии**: Версия мода.
- **Генератор кодов**: Создаёт фейковый промокод.

## 📥 Установка

1.  **Установи Tampermonkey**:
    - [Chrome / Edge](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
    - [Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
    - [Opera](https://addons.opera.com/en/extensions/details/tampermonkey-beta/)

2.  **Установи скрипт**:
    - [готовое решение](https://greasyfork.org/ru/scripts/567164-csgo-clicker-mod-menu)
    - Создай новый скрипт в Tampermonkey и вставь код из `mode.js`.
    - Опционально: опубликуй/установи через Greasy Fork (если загрузил туда).

3.  **Запуск**:
    - Открой [CSGO Clicker](https://csgo.mtsl.dk/).
    - Нажми **`N`** — открыть/закрыть меню.
    - Нажми **`J`** — включить/выключить оверлей.

## 🎮 Управление

| Клавиша | Действие |
| :--- | :--- |
| **N** | Открыть / закрыть мод-меню (можно перетаскивать) |
| **J** | Включить / выключить оверлей статистики (можно перетаскивать) |
| **Esc** | Отмена активного выбора (например, “Дюп конкретного предмета”) |

## 🛠 Решение проблем

- **Меню не открывается?**
  - Проверь, что Tampermonkey включён.
  - Обнови страницу (F5).
  - Убедись, что скрипт активен для `csgo.mtsl.dk`.

- **Оверлей не обновляется?**
  - Выключи/включи его кнопкой **`J`**.
  - Убедись, что игра полностью загрузилась.

- **Ошибка “Item not found”?**
  - Название предмета должно совпадать точно с тем, что в игре.
  - Нажми **Удалить фейк-предметы**, чтобы починить инвентарь.
